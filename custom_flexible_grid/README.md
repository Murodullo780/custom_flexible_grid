# Custom Flexible Grid

[![pub package](https://img.shields.io/pub/v/custom_flexible_grid.svg)](https://pub.dev/packages/custom_flexible_grid)

A customizable, flexible grid widget for Flutter apps.  
Easily build staggered, multi-column layouts with adjustable spacing and smooth scroll behavior.

---

## ✨ Features

- Flexible vertical or horizontal layout
- Cross-axis and main-axis spacing
- Static list or builder support
- Scroll controller, physics, padding support
- Clean and extensible API

---

## 🚀 Installation

Add this to your `pubspec.yaml`:

```yaml
dependencies:
  custom_flexible_grid: ^1.0.2

```
